# 🏦 Loan Approval Prediction System (ML Web App)

## 📌 Project Overview

The Loan Approval Prediction System is a Machine Learning-based web application developed using Flask that predicts whether a loan application will be approved or rejected based on user input.

The system uses a trained Decision Tree Classifier model to analyze key financial factors such as age, income, loan amount, and credit score. It provides real-time prediction along with approval probability.

This project demonstrates the integration of Machine Learning with Web Development and represents an end-to-end ML deployment pipeline.

---

## 🎯 Objective

The main objectives of this project are:

- To automate the loan approval decision process
- To reduce manual evaluation effort in financial institutions
- To provide quick and data-driven predictions
- To demonstrate ML model deployment using Flask

---

## 🧠 Machine Learning Model

- Algorithm Used: Decision Tree Classifier
- Type: Supervised Learning (Classification)
- Framework: Scikit-learn
- Output: Binary Classification (Approved / Rejected)

The model is trained on historical loan data and learns patterns based on financial attributes of applicants.

---

## 📊 Dataset Description

The dataset includes the following features:

- Age → Age of the applicant
- Income → Annual income of applicant
- LoanAmount → Requested loan amount
- CreditScore → Creditworthiness score (300–900)
- Approved → Target variable (1 = Approved, 0 = Rejected)
